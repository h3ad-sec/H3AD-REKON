# H3AD-REKON

**Attack Surface Reconnaissance — Part of [H3AD-SEC](https://h3ad-sec.github.io)**

> Attack Surface · Recon · Exposure Intelligence · Threat Actor Infrastructure

H3AD-REKON is a Shodan-powered reconnaissance platform for attack surface intelligence, CVE discovery, C2 hunting, and exposure mapping.

Live at: [h3ad-sec.github.io/H3AD-REKON](https://h3ad-sec.github.io/H3AD-REKON/)

## Features

- Host/IP lookup with reputation and exposure context
- Attack surface footprint mapping
- CVE and vulnerability discovery
- C2 infrastructure hunting
- Exposed services and asset discovery
